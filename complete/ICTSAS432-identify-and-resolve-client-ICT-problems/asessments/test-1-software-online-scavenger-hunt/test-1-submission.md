In your own words summarize the purpose of software. Give examples. 
Selected Answer: 	
```
The main purpose of software is to provide a layer between computer hardware and a user to make the computer useable by humans.

There are many types and 'layers' of software, but these can be broadly divided int:

*    system software; that is software that provides a platform to interact with the physical hardware, this is typically bundeled as an operating system, and includes device drivers, system services (display managers and the like), as well as the operating system kernel.
*    application software, which provides is make it possible for human users to interact with the computer and perform tasks, examples include word processors and publishing tools, IDE's of coding, video player and games for entertainment.
```

The "best" productivity tool is the one that is least intrusive, easiest to use and most effectively tracks the task or thing that you want tracked.

Which specific tool that is, very much depends on what is being tracked!

If I was looking to track how many steps I took in my day, for me personally, I would probably use the in-built google or samsung app in my phone - but I wouldn't reccommend that for an i-phone user!

However when I want to track how many tapes have been read for our current transcription project in Malaysia, I use Power BI to create a report based on the information in our database. That would not be a good use of my time if I was tracking household cleaning chores though.

Choosing the best best productivity software requires considering what is being tracked, what information needs to be collected (and for what purpose) and choosing a tool that is a appropriate.
