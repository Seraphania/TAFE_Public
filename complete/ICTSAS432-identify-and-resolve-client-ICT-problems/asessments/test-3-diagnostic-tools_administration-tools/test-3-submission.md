### Question 10
Complete  

**Disk Management**
1. [Open Computer Management](https://support.microsoft.com/en-us/windows/disk-management-in-windows-ad88ba19-f0d3-0809-7889-830f63e94405) and select Disk Management
2. Right click C: drive and shrink drive
3. An allocated space will appear
4. Right click the unallocated space
5. Select New Simple Drive and give the drive a letter
6. Select file System: NTFS, Allocation unit size: default, and Volume Label: Storage
7. Select Finish
8. Open File Explorer and view the new Drive

- [x] Complete
