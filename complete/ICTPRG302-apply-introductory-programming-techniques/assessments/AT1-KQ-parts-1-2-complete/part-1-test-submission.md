### Question 20
Needs Marking  
Unmarked 	

**Explain how sequence, selection (using if statements), and iteration (using loops) control the order in which a program's code is executed.**  
Selected Answer: 	
```
Sequence is the basic structure that code is executed in: each line one at a time, from top to bottom. This is easy to understand and follow, but limited.

Selection allows certain parts of code to only be executed under certain conditions, if a condition is not met then that part of the code block will be skipped. This makes it possible for the code to "make decisions" and execute only certain code.

Iteration allows for repetition of tasks, either indefinitely or a certain number of times. This reduces the amount of code that needs to be written and makes it easier to read and understand. the block is still read top too bottom, but will go back to the "for" or "while" until the exit condition is met.
```
